## Requirement
You should add api key to gradle.properties before build app.
`NEWS_API_KEY="YOURS_API_KEY"`

## Teck Stack

* Kotlin
* Jetpack Compose
* Material Design
* Hilt 
* Coroutines
* Flow
* AndroidX
* Retrofit
* SOLID Principle
